# Claims Review (Public Record)

This repository contains a structured, claim-by-claim factual review of public reporting related to Running Tide.

Maintained by Marty Odlin in a personal capacity to clarify the factual record.

Contents:
- Machine-readable claims inventory (CSV)
- Written appendices
- Primary source exhibits

This repository does not seek to litigate intent or editorial motives.
It exists to document and clarify facts with supporting evidence.
